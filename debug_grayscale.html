Grayscale Debug Page

Visit the main app at http://localhost:5183 and check if:

  1. The "显示灰度信息" checkbox is checked in the right panel
  2. Mouse over the video/image shows grayscale values
  3. Check browser console for JavaScript errors (F12)

How to Enable Grayscale Display:

  1. Upload a video file
  2. Select a ROI region
  3. In the right panel, find "灰度值分析" section
  4. Check the "显示灰度信息" checkbox
  5. Move mouse over the video to see grayscale values

Expected Behavior:

  • Grayscale value should appear next to "ROI Selected" button
  • Should show format: "灰度: XXX/255"
  • Value should update as mouse moves
